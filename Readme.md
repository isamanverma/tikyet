
# Tikyet Todo List

Tikyet is a simple and intuitive Todo List application that helps you stay organized and manage your tasks effectively. With Tikyet, you can easily create, track, and complete your tasks, ensuring nothing falls through the cracks.

## Features

- Create new tasks with a title and optional description.
- Mark tasks as completed to track your progress.
- Edit task details to add or modify task information.
- Delete tasks that are no longer needed.
- Filter tasks based on their completion status.
- Responsive design for a seamless experience across devices.

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

To run the Tikyet Todo List app locally, follow these steps:

1. Clone the repository:

   ```
   git clone https://github.com/isamanverma/tikyet.git
   ```

2. Navigate to the project directory:

   ```
   cd tikyet
   ```

3. Open the `index.html` file in your preferred web browser.

4. Start managing your tasks with Tikyet!
